# Gerador de casos de teste

Desenvolvimento de um programa em SML que gere casos de teste para o trabalho da disciplina de Estrutura de Dados.  <br />

O programa gera 3 tipos de arquivo: <br />  <br />
.cro: Contem as cordenadas das quadras, hidrantes, torres, semaforos.<br />
.pol: Contem as cordenadas que pertencem ao poligno.<br />
.svg: Amostra visual das quadras, hidrantes, torres, semaforos e polignos gerados. <br /> <br />
